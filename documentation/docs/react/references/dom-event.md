# DOM Events

| Event         |                                                                                |
| ------------- | :----------------------------------------------------------------------------- |
| **on_change** | The onchange event occurs when the text of the page is changed                 |
| **on_select** | The onSelect event occurs when user selects or highlights the text of the page |
| **value**     | To assign a text to a page                                                     |
